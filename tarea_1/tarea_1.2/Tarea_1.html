Definición de HTML

HTML es un lenguaje de marcado utilizado para crear páginas web. Significa HyperText Markup Language (Lenguaje de Marcado de Hipertexto). HTML se utiliza para estructurar el contenido de una página web, como texto, imágenes, enlaces y otros elementos multimedia. Es la base de la mayoría de los sitios web y se combina con CSS y JavaScript para crear experiencias web interactivas y atractivas.

CSS es un lenguaje de estilo utilizado para describir la presentación de un documento HTML. Permite controlar el diseño, la tipografía, los colores y el diseño general de una página web.

JavaScript es un lenguaje de programación utilizado para crear experiencias web interactivas. Se utiliza para agregar funcionalidad dinámica a las páginas web, como formularios, animaciones y manipulación del DOM.
